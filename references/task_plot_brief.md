# Task Plot Brief

## Task

- Title: Tapping Synchronization Task
- Construct: sensorimotor synchronization / auditory-motor timing
- Paradigm: tap with metronome beats, then continue tapping from memory.

## Rows

- Practice 600 ms: one practice trial.
- Test 450 ms: scored tempo condition.
- Test 600 ms: scored tempo condition.
- Test 750 ms: scored tempo condition.

## Trial Timeline

1. Ready: 800 ms ready screen before each trial.
2. Sync tapping: 10 metronome beats. Press SPACE in time with the tone.
3. Continuation tapping: 30 silent beats. Continue pressing SPACE at the remembered tempo.
4. Practice break or test ITI: practice break is 1200 ms; test ITI is 800 ms.

## Notes

- Metronome tone is 1319 Hz and 100 ms.
- There are no points, rewards, or correctness feedback.
- The task measures sync asynchrony, continuation interval mean/CV, and omissions.
